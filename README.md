# DiLoCo Calculator
An interactive calculator based on *trilogy* of DiLoCo papers: 
1. [DiLoCo: Distributed Low-Communication Training of Language Models](https://arxiv.org/abs/2311.08105)
2. [Streaming DiLoCo with overlapping communication: Towards a Distributed Free Lunch](https://arxiv.org/abs/2501.18512)
3. [Communication-Efficient Language Model Training Scales Reliably and Robustly: Scaling Laws for DiLoCo
](https://arxiv.org/abs/2503.09799)
